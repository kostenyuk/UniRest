Справочник «Шаблоны информационных карт»

Настройка шаблонов пластиковых карт осуществляется по следующим правилам:
  Реквизиты:
   Наименование - название шаблона;
   Шаблон кода - шаблон кода карты записанного на магнитной полосе;
   Шаблон номера - шаблон номера КК формируемый из номера карты записанного на магнитной полосе;
   Ограничения непреобразованого номера - ограничения накладываемые на номер карты записанного на магнитной полосе.
   
   Шаблон кода - шаблон кода карты записанного на магнитной полосе из которого система извлекает по указанным в шаблоне правилам номер карты.
   В шаблоне допускается  использование следующих служебных символов:
    % - любой символ (обязательный - в случае его отстутсвия код карточки признается не отвечающим требованияем шаблона);
    N - очередная цифра извлекаемого номера карточки (обязателная - в случае ее отстутсвия код карточки признается не отвечающим требованияем шаблона);
    n - очередная цифра извлекаемого номера карточки (не обязательная - в случае ее отстутсвия к извлекаемому номеру никакая цифра не добовляется,
        код карточки признается удовлетворяющим требованияем шаблона).
  Примеры шалонов кода:
   ;778=05010000=NNNNNNNN - КК (не локальная, восми значная);
   ;778=05010000=NNNNNNnn - КК (не локальная, 6 - 8 значная);
   ;778=0501%%%%=NNNNNN - КК (локальная, шести значная);
   ;778=05020000=80NNNNNN - КК (локальная ЛЕОКЛУБ, восми значная);
   ;4138320%NNNNNNN%= - пластиковая карта VISA Electron банка "Аваль";
   ;4138300%NNNNNNN%= - пластиковая карта VISA Standard банка "Аваль";
   ;4138301%NNNNNNN%= - пластиковая карта VISA Silver банка "Аваль";
   ;4138310%NNNNNNN%= - пластиковая карта VISA Gold банка "Аваль";
   ;422029%%%NNNNNN%= - пластиковая карта банка "РОДОВИД";
   ;422030%%%NNNNNN%= - пластиковая карта банка "РОДОВИД".
  Шаблон номера - шаблон номера КК формируемый из номера карты записанного на магнитной полосе по указанным в шаблоне правилам.
   В шаблоне допускается  использование следующих служебных символов:
    N - очередная цифра номера КК (обязателная - в случае ее отстутсвия к номеру КК добаляется ноль);
    n - очередная цифра номера КК (не обязателная - в случае ее отстутсвия к номеру КК не добаляется ничего).
  Примеры шалонов номера:
   NNNNNNNN - КК (восми значная);
   NNNNNNnn - КК (6 - 8 значная);
   NNNNNN - КК (шести значная);
   80NNNNNN - КК (локальная ЛЕОКЛУБ, восми значная);
   4NNNNNNN - пластиковая карта VISA Electron банка "Аваль";
   5NNNNNNN - пластиковая карта VISA Standard банка "Аваль";
   6NNNNNNN - пластиковая карта VISA Silver банка "Аваль";
   7NNNNNNN - пластиковая карта VISA Gold банка "Аваль";
   99NNNNNN - пластиковая карта банка "РОДОВИД".
             Ограничения непреобразованого номера - ограничения накладываемые на номер карты записанного на магнитной полосе. Ограничение накладывается диапазоном значений.
  В случае наличия нескольких ограничений система выполняет объединения диапазонов т.е. выполняет операцию "ИЛИ".
  Интерпретация операций:
   Номер Є [X, Y]  -  X <= Номер <= Y;
   Номер Є (X, Y]  -  X <   Номер <= Y;
   Номер Є [X, Y)  -  X <= Номер <  Y;
   Номер Є (X, Y)  -  X <   Номер <  Y.
  Примеры ограничения непреобразованого номера:
   КК скидка 5% - Номер Є [85000000, 85999999],
   КК скидка 10% - Номер Є [10000000, 10999999];
   КК скидка 15% - Номер Є [15000000, 15999999],
   КК скидка 20% - Номер Є [20000000, 20999999],
   КК скидка 22% - Номер Є [22000000, 22999999];
   КК банкетная скидка 25% - Номер Є [25000000, 25999999].
   Пластиковая карта банка "РОДОВИД" скидка 12% - Номер Є [280000, 479999].
   Пластиковая карта банка "РОДОВИД" скидка 15% - Номер Є [580000, 679999].
   Пластиковая карта банка "РОДОВИД" скидка 17% - Номер Є [550000, 579999].
   Пластиковая карта банка "РОДОВИД" скидка 22% - Номер Є [710000, 719999].
