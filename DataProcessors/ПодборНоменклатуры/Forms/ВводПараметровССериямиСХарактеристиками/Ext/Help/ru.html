Форма ввода количества и цены при подборе номенклатуры

В форму вводится информация о количестве и цене тех товаров, которые подбираются в документ.

В том случае , если установлен вид подбора "По справочнику" появляется дополнительный список для ввода товаров с характеристиками. При нажатии на кнопку "Заполнить" список заполняется теми характеристиками товаров, которые зарегистрированы для товара в справочнике "Характеристики номенклатуры".

В поле "Количество" необходимо ввести количество товаров с характеристикой, которое необходимо добавить в документ. Поле "Цена" заполняется значением цены для товара с характеристикой, которое зарегистрировано в регистре сведений "Цены номенклатуры". при необходимости цену можно изменить.

Если в диалоговом окне подбора установлен флаг "Серия", то дополнительно можно указать серию позиции номенклатуры.

Внимание. Серия и характеристика будут запрашиваться только в том случае, если для позиции номенклатуры установлены соответствующие флаги "Вести учет по сериям" и "Вести учет по доп.характеристикам".

  В том случае, если в программе подключены электронные весы, то с помощью кнопки "Получить вес"  можно считать вес товара.
