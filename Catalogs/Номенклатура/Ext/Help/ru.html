Справочник "Товары"

Справочник товаров предназначен для хранения информации о блюдах, модификаторах, комплектах, услугах.

Реквизиты шапки справочника:

  • Полное наименование. Используется, например, для вывода в печатные формы документов.
  • Краткое наименование. Используется при печати на кассовый аппарат.
  • Артикул. Текстовое значение, используется в печатных формах документов.

На закладке «Основные» указывается:

  • Тип товара
  • Товарная группа. Одна позиция номенклатуры может быть отнесена только к одной товарной группе, соответственно одна товарная группа может состоять из многих позиций номенклатуры.
  • Штучный. Признак штучного блюда.
  • Весовой. Признак весового блюда.
  • Временной. Признак временного блюда.
  • Уточняемый. Признак уточнения цены при добавлении блюда в заказ.
  • Модифицируемый. Признак модифицируемого блюда.
  • Максимальное количество. Максимальное количество модификатаров добавляемых в заказ для данного блюда.
  • Множественно. Признак множественного выбора модификаторов.
  • Допуская повторение. Признак возможного повторения модификаторов добавляемых в заказ для данного блюда.
  • Печатаемый. Признак вывода на печать в пречеке.
  • Безскидочный. Признак безскидочного блюда(скидки не распространяются.
  • Безнадбавочный. Признак безнадбавочного блюда(наценка не распространяется).

Закладка «Модификаторы»

В таблице «Модификаторы» представлен перечень модификаторов, флагами отмечены те, которые относятся к данному блюду.

Закладка «Условия»

В таблице «Условия» флагами отмечены условия приготовления, которые относятся к данному блюду.
