Установка параметра "Режим завершения работы пользователей"

Параметр устанавливается для управления работой пользователей в информационной базе. Для каждого пользователя с интервалом в 1 минуту проверяется, разрешено ли работать в ИБ.

Описание режимов работы пользователей.

Разрешить работу. Нормальный режим, при котором разрешена работа всех пользователей.

Предупредить пользователей о завершении работы. Пользователям* сообщается о необходимости завершить работу самостоятельно. Завершение работы системы в данном режиме не производится. Пользователь должен сам завершить работу системы.

Завершить с сохранением данных пользователя. Пользователям* сообщается о необходимости завершить работу, выдается предупреждение: "Работа системы будет завершена". Если у пользователя при этом открыты и модифицированны некоторые формы, задается вопрос о сохранении данных. Данная проверка производится каждую минуту.

Прекратить работу безусловно. В данном режиме производится прекращение работы системы всех пользователей*. Модифицированные и не записанные данные при этом не сохраняются.

Установить параметр "Режим завершения работы пользователей" может администратор или пользователь, обладающий правом  "Право завершения работы пользователей".

* Кроме пользователей обладающих правом  "Право завершения работы пользователей".
