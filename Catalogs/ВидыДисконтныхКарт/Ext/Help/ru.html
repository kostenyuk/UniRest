Справочник "Виды дисконтных карт"

В справочник "Виды дисконтных карт" вводится информация о видах дисконтных карт (обычная карта, серебряная карта, золотая карта и т.д.)

Данная информация используется при установке скидок по дисконтным картам.

Имеет следующие реквизиты:

  • Режим – определяет режим;
  • Условия – признак использования условий:
  • Диск – значение которое определяет вид карты по базе Козырных Карт;
  • Монетарность - признак депозитной карты;
  • Накопительная  - признак накопления:
  • Оплата - вид оплат для расчета депозитной картой;

Условия

В таблице «Условия» представлен перечень условий данного вида карт. Данная информация используется при установке скидок по дисконтным картам. Таблица доступна только если признак "Условия" имеет значение "Используется".
