Справочник «Типы  акций»

Справочник «Типы акций» предназначен для хранения списка типов акций, которые могут быть применены к документу.

 

  • Наименование – название типа акции.
  • Период действия - период, в течении которго будет действовать данная акция.
  • Номенклатура - товар, который предоставляется по данной акции.

 

В качестве условий акции могут быть заданы следующие условия:

  • По виду сертификата — это условие означает, что акция будет применена для соответствующего вида сертификата.
  • При создании заказа — это условие означает, что акция будет применена, если количество гостей при создании заказа первысило(либо равно) определенную величину(значение условия).
  • При закрытии заказа — это условие означает, что акция будет применена, если количество гостей при закрытии заказа первысило(либо равно) определенную величину(значение условия).
