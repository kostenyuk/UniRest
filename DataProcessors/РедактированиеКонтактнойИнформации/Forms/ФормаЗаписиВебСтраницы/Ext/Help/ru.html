Форма редактирования контактной информации. Веб-страница.

Форма предназначена для записи и редактирования веб-страниц объектов информационной базы.

В поле «Объект», выбирается объект информационной базы, для которого вводится веб-страница:

  • Контрагенты

  • Контактные лица

  • Физические лица

  • Организации

  • Пользователи

    При вводе новой веб-страницы из таблицы контактной информации в форме объекта, объект подставляется автоматически.


    Вид веб-страницы может быть 2-х типов:

    • Общий
    • Персональный

    Общий вид контактной информации.

    Общий вид контактной информации выбирается из справочника «Виды контактной информации». В данном справочнике хранятся виды контактной информации, которые присущи большинству объектов контактной информации одного типа (контрагеты, пользователи и пр.). По видам контактной информации в дальнейшем можно отобрать или сгрупприровать отчет о контактной информации.

    Персональный вид контактной информации.

    Персональный вид контактной информации определяется только для текущего объекта. При начале ввода наименования вида контактной информации программа производит подстрочный поиск в справочнике «Виды контактной информации». Если элемент (или элементы), соответствующий введенному наименованию, найден, то система ведет себя стандартно. Если найден единственный элемент вида контактной информации данного типа и выбранного типа объекта - то он автоматически подставляется. Если найдено несколько - то предлагается выбрать из списка. В обоих случаях выбранное значение вида контактной информации является общим. Если программа не нашла ни одного вида контактной информации по введенному наименованию, то цвет текста становится синим, это означает что данный вид контактной информации является персональным для текущего объекта.


    Пользователь может открыть введенную веб-страницу. Для отображения страницы используется стандартный браузер операционной системы.

    При нажатии кнопки ОК производится запись всех изменений в регистр сведений «Контактная информация».
